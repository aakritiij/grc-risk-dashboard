# Project: grc-risk-dashboard

This project aims to provide a comprehensive risk dashboard using Streamlit, allowing users to visualize and manage risks effectively.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/grc-risk-dashboard.git
   ```
2. Navigate to the project directory:
   ```
   cd grc-risk-dashboard
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

To run the Streamlit application, execute the following command:
```
streamlit run app.py
```
